About Us

Movie Review & Sentiment Analyzer is a web-based application developed as part of an AWS Capstone Project. The platform enables users to browse movies, submit reviews, and analyze audience sentiment using interactive visualizations.

Project Objective

The primary goal of this project is to demonstrate how real-time user feedback can be collected, processed, and visualized using modern web technologies and cloud infrastructure. It bridges the gap between theoretical learning and practical deployment.

Core Features

  • Browse movies with ratings and review counts
  • Submit user reviews and star ratings
  • Automatic sentiment classification (Positive, Neutral, Negative)
  • Interactive charts for sentiment and rating trends

Technology Stack

  • Frontend: HTML, CSS, JavaScript for responsive and user-friendly design
  • Backend: Python with Flask framework for application logic
  • Data Handling: In-memory Python data structures for managing reviews and movies

Cloud & AWS Integration

This project is designed with cloud scalability in mind and follows AWS best practices. The application architecture aligns with real-world cloud deployment standards.

  • Amazon EC2: Hosting the Flask web application
  • AWS IAM: Secure access and role-based permissions
  • AWS SNS: Event-driven notifications and messaging (conceptual integration)

Academic Relevance

This project serves as a learning platform for understanding full-stack development, cloud deployment concepts, and sentiment analysis. It is suitable for academic evaluation, demonstrations, and future enhancements.

Future Enhancements

Future versions of this application can include advanced machine learning models, real-time database integration, user authentication, and large-scale cloud deployment for handling high traffic.
